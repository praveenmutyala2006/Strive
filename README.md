# STRIVE

STRIVE is a behavioral productivity web platform focused on helping users build consistency, maintain momentum, and recover motivation through interactive habit tracking, behavioral analytics, and mood-based focus systems.

The website combines productivity tracking with motivational support features such as:
- weighted momentum scoring
- weekly behavioral analytics
- guided breathing
- sprint focus timers
- mood-based recommendations
- motivational audio and media systems
- dynamic behavioral insights

Built using HTML, CSS, and Vanilla JavaScript with a strong focus on UI design, behavioral logic, and immersive user experience.
